This application was tested on the following emulators and phones:
    1.  Pixel 3 API 27
    2.  Nexus 5X API 27
    3.  Samsung Galaxy S9
---------------------------------------------------------------------------------------
This application has the following features:
    1.  Current & Past Raffles
    2.  View Raffle Details
    3.  Create, Edit & Delete Raffle
    4.  Draw winner
    5.  Create Customer
    6.  Sell Raffle
    7.  View Sold Tickets
    8.  Share Ticket Information`
---------------------------------------------------------------------------------------
This application has the following know issue(s):
    1.  On back press the fragment does not resume, hence instead of using the back
        button use the Navigation menu to move across the application.
